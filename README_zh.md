# MTool JSON 翻译器

🌍 语言：
[English](README.md) | **简体中文** | [Français](README_fr.md)

一个基于 AI 的桌面应用程序，用于翻译 MTool 导出的 JSON 文件，支持本地大语言模型（LLM）以及兼容 OpenAI API 的服务。

## 计划功能

- 保持 JSON 结构不变，仅翻译文本内容
- 支持 LM Studio 与 OpenAI Compatible API
- 批量翻译
- 占位符保护（如 `{name}`、`%s`）
- 翻译缓存
- 术语表（Glossary）
- 桌面图形界面

## 技术栈

- Java 21
- Spring Boot
- Maven
- Jackson
- LM Studio
