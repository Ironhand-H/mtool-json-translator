# MTool JSON Translator

🌍 Language:
**English** | [简体中文](README_zh.md) | [Français](README_fr.md)

An AI-powered desktop application for translating MTool exported JSON files using local LLMs or OpenAI-compatible APIs.

## Planned Features

- Translate JSON values while preserving structure
- Support LM Studio and OpenAI-compatible APIs
- Batch translation
- Placeholder protection (`{name}`, `%s`, etc.)
- Translation cache
- Glossary support
- Desktop GUI

## Tech Stack

- Java 21
- Spring Boot
- Maven
- Jackson
- LM Studio

## Status

🚧 This project is currently under active development.
