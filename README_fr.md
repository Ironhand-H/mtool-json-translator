# Traducteur JSON pour MTool

🌍 Langue :
[English](README.md) | [简体中文](README_zh.md) | **Français**

Une application de bureau basée sur l'IA permettant de traduire les fichiers JSON exportés par MTool à l'aide de modèles de langage locaux (LLM) ou d'API compatibles OpenAI.

## Fonctionnalités prévues

- Traduction des valeurs JSON tout en conservant la structure
- Prise en charge de LM Studio et des API compatibles OpenAI
- Traduction par lots
- Protection des variables (`{name}`, `%s`, etc.)
- Cache de traduction
- Glossaire
- Interface graphique

## Technologies

- Java 21
- Spring Boot
- Maven
- Jackson
- LM Studio
